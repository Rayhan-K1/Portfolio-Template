# Portfolio Template

A portfolio website built for a client who had just completed a university degree in Games Programming.

All sensitive and personal details have been removed/redacted.

## Tech Stack

- **HTML5**, **CSS**, **JavaScript**
- **Vue.js 2** – used for reactive, component-based UI
- **Vue Router** – for client-side navigation
- **Bootstrap** – for layout and responsive styling
- **jQuery** – for AJAX requests
- **Underscore.js** – for utility functions
- **Cypress** – for end-to-end testing
- **Figma** – for application and UI design
